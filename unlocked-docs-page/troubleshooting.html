Troubleshooting

It often happens that you experience a problem while installing or using Arix Theme. Therefore, in this article we have explained for occurring error's of Arix with solutions belonging to them.

Tip, we've built a bot that can recognise pre-programmed errors and give you the right solution, which you can find on our Discord server.

First, you need to understand what kind of error you're experiencing. We divide them into 2 categories, front-end bugs and back-end bugs. You can recognise a front-end error when you either use the yarn build:production command, or you refresh your Pterodactyl panel and it either is a white screen or loads but tells you to reload/refresh the application. In all other cases it is probably caused by a backend error.

How to troubleshoot when using the installer?

We're now going to discuss the errors you may encounter when installing Arix Theme. It is important to read your error messages and not panic or do things without reading them!

  • 500 error after installing
  • White screen after installing
  • No theme after installing

500 error after installing

If you're experiencing a 500 error in most cases you can fix the error by running the commands below:

chmod -R 755 storage/* bootstrap/cache
php artisan migrate --force
php artisan optimize:clear
php artisan optimize 

chown -R www-data:www-data /var/www/pterodactyl/*
chown -R nginx:nginx /var/www/pterodactyl/*
chown -R apache:apache /var/www/pterodactyl/*

White screen after installing

Are you experiencing a white screen after installing Arix Theme? This is probably caused by one of the following errors.

Node.js version is v20 (>= 17)
Browserslist: caniuse-lite is outdated. Please run:
npx update-browserslist-db@latest
node:internal/crypto/hash:68
this[kHandle] = new _Hash(algorithm, xofLen);

Did the installer give this error? This is because you're running Pterodactyl on node v17+. Fix with the commands below:

export NODE_OPTIONS=--openssl-legacy-provider
php artisan arix install

Node.js version is v20 (>= 17)
error Command failed with exit code 2.

If you get a white screen after the installer says Command failed with exit code 2:

php artisan down
rm -rf /var/www/pterodactyl/resources
php artisan arix uninstall
php artisan arix install
php artisan up

No theme after installing

After installing Arix Theme, is the default Pterodactyl theme still visible?

If installer said:

no such file or directory: add

Then you installed incorrectly. Follow:

https://pterodactyl.io/community/customization/panel.html 
Node.js version is v16 (< 17)
RpcIpcMessagePortClosedError: Process exited [SIGKILL]
error Command failed with exit code 1

This means your server does not have enough RAM. Minimum recommended: 3 GB.
